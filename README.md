# Fish Tank

A fun landing page for your browser.

## Installation

```bash
yarn install
```

## Usage

```bash
yarn build
```
Then open dist/index.html in your browser.

## Dependencies
- [Tween.js](https://github.com/tweenjs/tween.js)
- [Fitty](https://github.com/rikschennink/fitty)
- [WebPack](https://webpack.js.org/)
